# 🚀 CI/CD Pipeline Project (AWS + Docker + Jenkins)

## 📌 Project Overview

This project demonstrates a complete CI/CD pipeline by deploying a containerized Flask application on AWS EC2 using Docker and Jenkins.

## 🧱 Architecture

Developer → GitHub → Jenkins → Docker → AWS EC2 → Live Application

## ⚙️ Technologies Used

* AWS EC2
* Docker
* Jenkins
* Python (Flask)
* GitHub
* Linux

## 🚀 Features

* Automated CI/CD pipeline using Jenkins
* Dockerized Flask application
* Live deployment on AWS EC2
* GitHub webhook integration (auto trigger)
